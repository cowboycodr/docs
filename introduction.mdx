**KianKit** is designed for those who like to move quick. It gets rid of all of the boring stuff. Allowing you to work on what matters.

## Features

- [**Svelte**](https://svelte.dev): Component-based JavaScript framework, that prioritizes speed and simplicity.
- [**SvelteKit**](https://kit.svelte.dev): An app framework built on top of Svelte. Provides routing, SSR, and more.
- [**Supabase**](https://supabase.com): An open-source firebase alternative with a generous free tier, and self-hosting options.
- [**Tailwind CSS**](https://tailwindcss.com): Utility-first CSS framework for rapidly building modern websites without ever leaving your HTML.
- [**Shadcn UI**](https://shadcn-svelte.com): Beautifully designed components that you can copy and paste into your apps. Accessible. Customizable. Open-source.
- [**Lucide**](https://lucide.dev): Open-source library that provides 1000s of beautiful & consistent icons.
- [**Moment.js**](https://momentjs.com): Parse, validate, manipulate, and display dates and times in JavaScript.
- [**Font awesome**](https://fontawesome.com): Font Awesome is the internet's icon library and toolkit. Provides up-to-date brand icons.
- [**Mode Watcher**](https://mode-watcher.svecosystem.com): SSR-friendly light and dark mode for SvelteKit.

## Motivation

KianKit was born out of the desire to make full-stack apps fast. To make web development less boring. So that we can experiment more.
